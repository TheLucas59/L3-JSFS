# Sellyours - Lucas Plé

## Lancement
Après avoir clôné ce dépôt, installez les dépendances :
```
npm install
```

Pour lancer le serveur de base de données :
```
mdkir dbData
mongod --dbPath dbData
```

Pour lancer le serveur web :
```
npm run start
```

Pour vous connecter sur le site :

http://localhost:300
